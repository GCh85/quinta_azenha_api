---

## O que é

Site da Quinta da Azenha, uma quinta produtora de vinho Arinto em Bucelas.  
Tem 4 páginas: Início, Vinhos, Experiências e Contacto.

---

## Tecnologias usadas

- HTML5
- Bootstrap 5.3 (layout, navbar, cards, responsividade)
- CSS próprio (hero, cores, fontes, FAQ, erros do formulário)
- JavaScript puro (sem frameworks)
- Google Fonts - Playfair Display + Inter

---

## O que o JavaScript faz

- Ano automático no footer
- FAQ acordeão (abre/fecha perguntas)
- Filtro de vinhos por tipo (Branco / Tinto / Espumante)
- Validação do formulário de contacto (nome, email, assunto, mensagem)

---

## Como abrir

Abrir o ficheiro `index.html` no browser. Não precisa de servidor nem instalações.
